# Keras

This project contains tasks related to building, training, and evaluating
neural networks using the Keras library (TensorFlow backend).

## Topics Covered

- Building sequential and functional Keras models
- L2 regularization and Dropout
- Compiling models with optimizers and loss functions
- Training with validation data and callbacks
- Saving and loading models and weights
- Using early stopping and learning rate decay
- Making predictions with trained models
- One-hot encoding labels

## Requirements

- Python 3.9
- TensorFlow / Keras 2.x
- NumPy 1.25.2
- Ubuntu 20.04 LTS

## Files

| File | Description |
|------|-------------|
| 0-sequential.py | Build a Keras model using the Sequential API |
| 1-input.py | Build a Keras model using the Input class |
| 2-optimize.py | Set up Adam optimizer with categorical crossentropy |
| 3-one_hot.py | One-hot encode labels |
| 4-train.py | Train a Keras model |
| 5-train.py | Train with validation data |
| 6-train.py | Train with early stopping |
| 7-train.py | Train with learning rate decay |
| 8-train.py | Train and save the best model |
| 9-model.py | Save and load a Keras model |
| 10-weights.py | Save and load model weights |
| 11-config.py | Save and load model configuration |
| 12-predict.py | Make predictions with a Keras model |
